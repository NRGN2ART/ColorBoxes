Analog Redbox by L.K.
=====================

<p> Design uses XR2240 to pulse dual oscilator circuits.

<p> The picture shows a version with three XR2240's that generates nickle, dime, and quarter

<p> 1700 + 2200 Hz
<p> Timing: Nickle { 66ms on 66ms off } , Dime { 66ms on 66ms off } X 2, Quarter { 33ms on 33ms off } X 5</p>
